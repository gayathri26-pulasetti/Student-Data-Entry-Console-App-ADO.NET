# Student Data Entry Console Application

## 📌 Project Overview
A simple C# console application that stores student details in a SQL Server database using ADO.NET. The project demonstrates basic database connectivity and secure data insertion using parameterized queries.

## 🚀 Features
- Accepts student details through console input
- Connects to SQL Server database
- Inserts records into Student table
- Uses parameterized queries to prevent SQL injection

## 🛠️ Tech Stack
- Language: C#
- Framework: .NET Console Application
- Database: SQL Server
- Data Access: ADO.NET

## ⚙️ Key Concepts Implemented
- ADO.NET Database Connectivity
- SQL INSERT Operations
- Parameterized Queries
- Handling User Input in Console

## 📥 Input Fields
- Student Number  
- Student Name  
- Student Address  
- Student Marks  

## 🗄️ Database Structure
- StuNo (INT)  
- StuName (VARCHAR)  
- StuAddress (VARCHAR)  
- StuMarks (INT)  

## ▶️ How to Run
1. Open the project in Visual Studio
2. Ensure SQL Server is running
3. Update connection string if required
4. Run the application
5. Enter student details in console
